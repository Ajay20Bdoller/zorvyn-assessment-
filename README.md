# Overview

This project is a modern Finance Dashboard UI built using React + Vite + TypeScript. It focuses on clean UI design, reusable components, and interactive data visualization.

The dashboard provides insights into financial metrics using charts, state management, and responsive layouts.

## Tech Stack

Frontend Framework: React
Build Tool: Vite
Language: TypeScript
Styling: Tailwind CSS
State Management: Zustand
Charts & Visualization: Recharts

### Features

Interactive financial charts
Data visualization using Recharts
Fast performance with Vite
Clean and modern UI design
Fully responsive layout
Global state management using Zustand
Modular and reusable components

## Installation & Setup

#### Clone the repository

git clone <your-repo-url>
cd zorvyn-assessment

#### Install dependencies

npm install

##### Run development server

npm run dev

##### App will run at:

http://localhost:5173

Vite
